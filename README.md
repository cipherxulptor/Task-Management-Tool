
# Task Management System

A full-stack task management application implementing comments, file attachments, and analytics with a secure REST API and a TypeScript-based React frontend.

---

## Backend Implementation

### Comments & Collaboration

* Task-level comment system with retrieval, update, and deletion support
* Proper authorization checks on all comment operations

### File Handling

* Multi-file uploads per task
* Secure file download and deletion

### Analytics & Reporting

* Aggregated task statistics by status and priority
* User performance metrics
* Task trend analysis over time
* Exportable task data for reporting

### Engineering & Architecture

* RESTful API design with correct HTTP semantics
* Structured error handling with Authentication and role-based authorization on protected routes
* Well-defined database schema with relationships and indexing
* Security controls including CORS, rate limiting, and input sanitization

---

## Frontend Implementation

### Core Features

* Authentication flow (login and signup)
* Dashboard with real-time overview statistics
* Detailed task views with editing capabilities
* User profile management
* Analytics and reporting interface

### UX & Performance

* Responsive layouts across screen sizes
* Form validation with clear error feedback

### Technical Stack

* React with hooks and TypeScript
* Client-side routing and state management
* Mongo-db
